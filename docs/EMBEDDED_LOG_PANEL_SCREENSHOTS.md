## 关键特性说明

### 🎨 彩色日志

在实际界面中，日志会用不同颜色显示：

- **INFO** (黑色): `2025-10-28 20:05:30 - INFO - 开始生成草稿`
- **WARNING** (橙色): `2025-10-28 20:05:39 - WARNING - 音频文件较大`
- **ERROR** (红色): `2025-10-28 20:05:45 - ERROR - 下载失败`
- **DEBUG** (灰色): `2025-10-28 20:05:50 - DEBUG - 调试信息`

### 🔧 可拖拽分隔线

```
═══════════════════════════════════════════
        ⬍ 拖拽这里调整上下比例 ⬍
═══════════════════════════════════════════
```

- **向上拖**: 日志面板变大，方便查看更多日志
- **向下拖**: 输入区域变大，方便输入更多内容

### 🔄 自动滚动

日志面板右下角会自动滚动到最新的日志行，除非用户手动向上滚动查看历史日志。

### 📱 响应式布局

窗口大小改变时，所有组件会自动调整：
- 输入框自动填充可用空间
- 日志面板保持可见且可用
- 按钮和标签保持正确位置

## 菜单选项

```
┌─ 查看(V) ────────────────────┐
│ ✓ 切换日志面板               │
│   日志窗口（独立）           │
└──────────────────────────────┘
```

## 实际使用截图

当您运行 `python src/main.py` 时，您会看到：

1. **主窗口尺寸**: 900x700 像素（比之前的 800x600 稍大）
2. **上部区域**: 约占 75% 空间，包含输入和控制
3. **下部区域**: 约占 25% 空间，显示日志
4. **分隔线**: 可拖拽调整比例
5. **日志面板**: 默认显示，可随时隐藏

## 使用建议

1. **首次使用**: 保持日志面板可见，熟悉生成过程
2. **输入大量内容**: 点击 [隐藏日志] 获得更多空间
3. **调试问题**: 向上拖拽分隔线，扩大日志显示区域
4. **多显示器**: 使用 "日志窗口（独立）" 选项
