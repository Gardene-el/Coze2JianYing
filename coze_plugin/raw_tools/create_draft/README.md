# create_draft

## 工具名称
`create_draft`

## 工具介绍
此工具对应 FastAPI 端点: `/create`

创建剪映草稿, 并指定其基本参数如分辨率、帧率等 草稿创建完成后, 可通过添加轨道和片段来构建完整的视频项目

## 输入参数

| 参数名称 | 参数描述 | 参数类型 | 是否必填 |
|---------|---------|---------|---------|
| draft_name | 项目名称 | str | 否 |
| width | 视频宽度（像素） | int | 否 |
| height | 视频高度（像素） | int | 否 |
| fps | 帧率 | int | 否 |
| allow_replace | 是否允许替换同名草稿 | bool | 否 |

## 输出参数

| 参数名称 | 参数描述 | 参数类型 | 是否必填 |
|---------|---------|---------|---------|
| draft_id | 返回创建的草稿ID | str | 是 |

## 使用说明
此工具由脚本自动生成，用于在 Coze 平台中调用对应的 API 端点。

工具会：
1. 生成唯一的 UUID
2. 记录 API 调用到 `/tmp/coze2jianying.py` 文件
3. 返回包含 UUID 的响应

## 注意事项
- 此工具在 Coze 平台的沙盒环境中运行
- API 调用记录保存在 `/tmp/coze2jianying.py`
- UUID 用于关联和追踪不同的对象实例
